# E-Commerce Website

A fully responsive e-commerce UI built during my Web Development Internship at **AgileITech, Hubli, Karnataka** as part of my 6th Semester.

## 🌐 Live Demo
[View the Live Site here!](https://AshwiniMY.github.io/ecommerce-site/home.html)

## 🚀 Features
- **Responsive Design**: Fully responsive UI ensuring seamless compatibility across mobile, tablet, and desktop devices.
- **Image Carousel**: Automated image carousel implemented using client-side DOM manipulation.
- **Real-Time Product Search**: Interactive product search powered by JavaScript DOM manipulation.
- **Persistent User State**: Browser `localStorage` used to persist dynamic user state data (e.g., cart items) across multiple pages.
- **Multi-Page Layout**: Home, Shop, Product Detail, Cart, Blog, About, and Contact pages.

## 🛠️ Technology Stack
| Technology | Usage |
|---|---|
| **HTML5** | Semantic structure and layout |
| **CSS3** | Styling, animations, and responsive design |
| **Bootstrap** | Responsive grid system and UI components |
| **JavaScript** | DOM manipulation, localStorage, interactivity |

## 📁 Project Structure
```
ecommerce-site/
├── home.html          # Landing page with carousel
├── shop.html          # Product listing with real-time search
├── product1-6.html    # Individual product detail pages
├── cart.html          # Shopping cart (localStorage powered)
├── blog.html          # Blog listing page
├── about.html         # About us page
├── contact.html       # Contact page
├── style.css          # Main stylesheet
└── Images/            # Product and banner images
```

## 💻 Local Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/AshwiniMY/ecommerce-site.git
   ```
2. Open the project folder.
3. Open `home.html` in any web browser — no local server required!
