# E-Commerce Website

A responsive, static e-commerce front-end application built with HTML, CSS, and Vanilla JavaScript. 

## 🌐 Live Demo
[View the Live Site here!](https://AshwiniMY.github.io/ecommerce-site/)

## 🚀 Features
- **Modern UI/UX**: Clean, attractive, and user-friendly interface.
- **Product Pages**: Individual details and listings for multiple products.
- **Shopping Cart**: Cart layout and design structure.
- **Responsive Design**: Adapts beautifully to mobile, tablet, and desktop screens.
- **Multiple Pages**: Home, Shop, Blog, About, Contact, and Cart pages.

## 🛠️ Technology Stack
- **HTML5**: For structural layout and semantic design.
- **CSS3**: For styling, responsive layouts, and animations.
- **JavaScript**: For interactive UI elements.

## 📁 Project Structure
- `home.html` / `index1.html` - Main landing page.
- `shop.html` - Main product listing page.
- `product[1-6].html` - Individual product detail pages.
- `cart.html` - Shopping cart UI.
- `blog.html` - Blog listing page.
- `about.html` - About us page.
- `contact.html` - Contact page.
- `style.css` - Main stylesheet containing all UI tokens.

## 💻 Local Setup
1. Clone the repository: `git clone https://github.com/AshwiniMY/ecommerce-site.git`
2. Open the project folder.
3. Open `home.html` or `index1.html` in any web browser. No local server required!
